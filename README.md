# RaceTournamentBracket
A tournament bracket app for racing mousetrap/rubber band cars that automatically manages both the winners bracket and losers bracket.

Requirments for project:
Core Features

1. Adjustable Bracket Size

The number of teams or players should be adjustable

The bracket should work for different tournament sizes

The user should be able to choose how many total participants are in the tournament


2. Name Input

The user should be able to copy and paste participant names

The app should automatically place those names into the bracket

It should support both individual players and teams


3. Automatic Randomization

The app should automatically randomize all participants at the start

After randomizing, it should place them into the bracket in bracket order




4. Winners Bracket Logic

If a participant wins, they move forward in the winners bracket

Winners should face the next winning participant in the next round

Each bracket or matchup should be clearly numbered


5. Losers Bracket Logic

If a participant loses, they should automatically move into the losers bracket

The losers bracket should continue eliminating participants as they lose again

The losers bracket should eventually allow a surviving participant to match up against the winners bracket finalist


6. Championship Structure

The app should allow the losers bracket winner to face the winners bracket winner

The user should be able to choose the format for the final winner bracket setup

This could include selecting how many wins are needed in the championship round


7. Placement Tracking

The app should organize results from 1st place to last place

Placement should be based on the order of elimination

The last undefeated person should be 1st

Earlier eliminations should rank lower automatically


8. Matchup Display

Every bracket should be numbered

The system should clearly show who is racing against who

It should be easy to follow both the winners and losers sides


Possible User Flow

User enters or pastes participant names

User selects bracket size

App randomizes participants

App creates winners bracket

Losers automatically drop into losers bracket after each loss

Winners keep advancing in winners bracket

Losers bracket continues until one remains

Losers bracket finalist faces winners bracket finalist

Final standings are generated from 1st to last
